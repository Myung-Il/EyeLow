파이썬 기본 문법

세미콜론 사용법

파이썬에서는 보통 한 줄에 하나의 구문을 작성해야 합니다

> print('Hello, world!')
결과 : Hello, world!

하지만 세미콜론(;)을 이용하면 여러 구문을 이어서 작성할 수 있습니다

> print('Hello, world!'); print('Hello, world!')
결과 : Hello, world!
결과 : Hello, world!

주석

주석은 하나의 줄을 설명란으로 만들 수 있습니다

> # Hello, world! 출력
> a = 1 + 2 # 더하기

주석 처리된 코드는 작동하지 않습니다

> # print('Hello, world!')

들여쓰기

들여쓰기는 코드를 읽기 쉽도록 일정한 간격을 띄워 작성하는 방법입니다

특히 파이썬은 들여쓰기가 문법입니다

> if a == 10:
> print('10입니다')
결과 : IndentationError: expected an indented block

올바른 들여쓰기 예제

> if a == 10:
>    print('10입니다')
결과 : '10입니다'
